# NarrIQ

## Overview

NarrIQ is an AI-powered SaaS platform that I designed and developed to automate qualitative market research workflows.

The main goal of NarrIQ is to reduce the manual effort required for analyzing large amounts of open-ended survey responses by using artificial intelligence and automation.

The platform helps market research teams process qualitative data faster by automating tasks such as response analysis, categorization, and sentiment understanding.

---

## Problem

In traditional market research workflows, analyzing open-ended survey responses requires significant manual effort.

Researchers usually need to read thousands of responses, manually assign categories, identify common themes, and analyze customer sentiment.

This process is time-consuming, difficult to scale, and can introduce inconsistencies between different analysts.

NarrIQ was created to solve this challenge by using AI-powered automation to accelerate qualitative analysis while maintaining accuracy.

---

## AI Pipeline

NarrIQ uses Large Language Models and semantic analysis techniques to automate qualitative text processing.

The AI workflow includes:

* Processing raw survey responses
* Generating semantic representations using embeddings
* Grouping similar responses using clustering techniques
* Performing sentiment analysis
* Automatically generating meaningful categories and insights

The system was designed to balance AI quality with operational cost by reducing unnecessary LLM API calls through intelligent response grouping.

## AI and Optimization

One of the main technical challenges was reducing the cost of LLM inference while maintaining high-quality results.

To solve this challenge, I implemented semantic embeddings and clustering techniques to group similar responses before sending them to the LLM.

This approach reduced the number of required API calls and improved processing efficiency for large datasets.
